安装文档

首次打开 微信助手 Pro 软件，会进入登录界面。

您需要点击注册按钮，填写授权码和 WeChat API License，并注册 “微信助手” 本地账号。



  • 授权码(Client Key)：需要联系 微信助手 Pro 客服获取。
  • WeChat API License：需要到 WeChat API 官网申请。

注册完成后，返回主界面，点击登录，会唤起微信登录页面，您需要扫描二维码登录微信。

等待数据加载完成，即可进入软件运行主界面，体验完整的 微信助手 Pro 功能。

当你需要退出登录时，点击右上角的退出按钮，即可退出登录。

注意：微信助手 Pro 软件退出登录后，微信登录也会退出。
