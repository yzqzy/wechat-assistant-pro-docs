# 智能对话

智能对话功能可以帮助你快速接入和使用聊天机器人，提升用户体验。

## 机器人

### 列表页

机器人列表，显示已配置的机器人以及基本信息。

<img src="/img/software_chatbot_01_mosaic.png" width="500" />

### 参数配置

支持选择 fastgpt、dify、openai 平台，配置相关参数后，即可使用机器人。

支持按照联系人、标签分组、群组设置不同的机器人，实现千人千面，不同场景的个性化服务。
