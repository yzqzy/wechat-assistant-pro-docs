# 群管理

群管理功能可以帮助您管理群组，包括创建、修改、删除群组，邀请或踢出成员，设置群公告、群头像和群名称等。

## 列表页

群聊列表，支持查看群聊名称、群成员、管理员、群公告、群头像等信息。

![图片](/img/software_chatroom_01_mosaic.png)

## 群成员管理

支持查看群成员基本信息、踢出群成员、设置群管理员等。

![图片](/img/software_chatroom_02_mosaic.png)

<br />

支持添加好友到群聊，邀请好友入群。
