# 群管理

群管理功能可以帮助你管理群组，包括创建、修改、删除群组、邀请成员、踢出成员、设置群公告、设置群头像、设置群名称等。

## 列表页

群聊列表，支持查看群聊名称、群成员、管理员、群公告、群头像等信息。

## 群成员管理

支持查看群成员基本信息、踢出群成员、设置群管理员等。
