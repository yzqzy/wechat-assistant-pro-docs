# 微信助手 Pro

**微信助手 Pro** 是一款专业的微信管理客户端，可以帮助您全面管理微信号，包括通讯录、群组、智能对话、朋友圈、定时提醒和直播间等功能。

## 产品特色

- 🚀 **独立部署** - 本地部署，数据安全可控
- 🛡️ **稳定可靠** - 稳定运行，不易被封号
- 🔄 **多开支持** - 支持同时登录多个微信号
- 📱 **功能全面** - 涵盖微信管理的各个方面

## 核心功能

### 📇 通讯录管理

- 查看好友信息（微信号、昵称、备注、头像、性别、地区等）
- 多种方式添加好友（群聊搜索、微信号/手机号搜索、附近的人、摇一摇）
- 删除好友、发送消息（支持文本、图片、文件、视频、音频等）
- 标签管理，对好友进行分类
- 导出通讯录和标签列表为 Excel 格式

### 👥 群组管理

- 创建、修改、删除群组
- 邀请或踢出群成员
- 设置群公告、群头像和群名称
- 查看群成员信息，设置群管理员
- 导出群聊列表和群成员列表

### 🤖 智能对话

- 支持接入多种 AI 平台（FastGPT、Dify、Coze、OpenAI 等）
- 按联系人、标签和群组设置不同的机器人，实现个性化服务
- Prompt 管理，支持系统内置和自定义 Prompt
- 支持通过回调地址接入自定义对话模型
- 聊天记录查看和导出功能

### 📢 群发助手

- 支持按联系人、标签和群组进行批量发送
- 灵活配置群发任务参数
- 高效的消息群发管理

### ⏰ 定时提醒

- 创建和管理定时任务
- 支持标签分组模式，更好地管理定时任务
- 查看任务执行状态和历史记录
- 导出定时提醒任务列表

### 🔔 回调事件

- 自定义关键字、用户类型等配置
- 提供丰富的功能扩展能力
- 支持自定义回调地址，灵活对接业务系统

### ⚙️ 系统设置

- 用户配置和系统配置
- 个性化设置选项

## 快速开始

### 系统要求

- Windows 系统电脑或云服务器
- 安装指定版本微信（3.9.11.25）
- 申请 [WeChat API](https://www.wechatsdk.com/zh) License

### 安装步骤

1. 联系客服获取 **微信助手 Pro** 软件和授权码
2. 申请 [WeChat API](https://www.wechatsdk.com/zh) License
3. 首次安装时输入授权码和 WeChat API License
4. 创建微信助手账号并登录
5. 扫描二维码登录微信，开始使用

详细安装文档请查看：[安装指南](/docs/getting-started/installation)

## 多开微信

**微信助手 Pro** 支持多开微信，可以同时登录多个微信号。通过指定不同的端口号，可以启动多个应用实例。

```bash
@echo off
start /b "" "D:\Install\wechat-assistant-pro\微信助手.exe" --port=4000
start /b "" "D:\Install\wechat-assistant-pro\微信助手.exe" --port=4002
start /b "" "D:\Install\wechat-assistant-pro\微信助手.exe" --port=8080
```
